# e-xgboost-web-accesibility
Engineered XGBoost model with SHAP explainability for automated WCAG 2.1 accessibility defect detection in DOM structures. Ghanaian web context study.

An engineered XGBoost model (E-XGBoost) for automated detection of 
WCAG 2.1 accessibility defects from DOM-extracted structural features 
of Ghanaian web interfaces.

- `notebooks/` — baseline and engineered model notebooks
